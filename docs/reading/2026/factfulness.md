# 《事实》 — Hans Rosling｜读书笔记

读完《事实》（Factfulness）这本书，最冲击我的不是书中复杂的宏观数据，而是开头那个看起来简单的“13 题常识小测验”。

结果非常打脸：**我只做对了 1 题，比随机瞎蒙的正确率（33.3%）还要低**。作者在书中打趣道，如果你把这些题目拿给动物园的黑猩猩做，它们扔飞镖选出来的正确率都比人类高。

这说明，我们对世界的认知不仅是过时的，甚至是系统性错误的。

我也把这套题目做成了一个在线链接，强烈建议你花 3 分钟测一下，看看你能否战胜黑猩猩： [事实测验 - 你对世界的认知，比大多数人更准确吗？](https://factfulness-quiz.elvinn.wiki/)

为什么我们会有如此大的认知偏差？作者汉斯·罗斯林结合他在联合国几十年的救援经历，总结了 10 个误导我们的“本能”。我认为可以将它们归纳为我们大脑的三大缺陷：

1. **喜欢过度简化（偷懒的大脑）**： 我们习惯线性思维（觉得人口会一直涨下去）、习惯单一视角（拿着锤子看什么都是钉子），甚至习惯刻舟求剑。
2. **容易感情用事（戏剧化的大脑）**： 我们总是关注负面新闻，对“正在发生的缓慢进步”视而不见；我们容易被个体的痛苦故事打动，却忽略了统计学上的群体改善。
3. **难以及时更新（迟钝的大脑）**： 我们脑海中的“第三世界”印象，可能还停留在几十年前的教科书里。

这本书虽然有不少统计数据，但因为作者分享了不少自身联合国工作和救援中的一些趣事，所以读起来一点也不枯燥。读完之后，有三个观点让我印象尤其深刻：

1. **事情可以既是糟糕的，又是正在变好的**。 就像早产儿在保温箱里虽然状态不好（糟糕），但各项指标都在提升（变好）。如果我们只盯着“糟糕”看，就会失去希望；如果只盯着“变好”看，就会盲目乐观。我们需要学会同时持有这两种视角。
2. **世界不是非黑即白的，绝大多数人生活在中间**。 我们习惯把世界分为“穷人”和“富人”、“发达国家”和“发展中国家”。但实际上，世界分为 4 个收入等级。在这个图谱中，极端贫困和极端富有都是少数，绝大多数人类其实都生活在中间状态，日子过得比我们想象中要好得多。
3. **注意“单一数字”的陷阱，永远在寻找对比**。 当你看到一个惊人的数字（比如“每年有 X 儿童死亡”），先别急着恐慌。问自己一个问题：“这个数字是多还是少？分母是多少？去年的数字是多少？” 孤独的数字不仅没有意义，而且往往具有欺骗性。

::: tip
在当下这个充斥着博人眼球消息的大环境中，保持“实事求是”的世界观，或许是我们维持客观、探寻真相最好的武器。
:::
